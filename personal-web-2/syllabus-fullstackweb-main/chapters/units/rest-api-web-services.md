# REST API Web Services

There are various popular web services that use REST API:

- [GitHub](https://developer.github.com)
- [Twitter](https://developer.twitter.com)
- [Facebook](https://developers.facebook.com)
  - [Graph API](https://developers.facebook.com/docs/graph-api)
  - [Facebook Graph Search Generator](http://graph.tips)
- [Instagram](https://instagram.com/developer)
- [Google Custom Search](https://developers.google.com/custom-search)
- [YouTube](https://developers.google.com/youtube)
- [Wikipedia](https://mediawiki.org/wiki/API:Main_page)
- [Zomato](https://developers.zomato.com)
- [Airbnb](https://airbnb.com/partner)
- [Mailchimp](https://developer.mailchimp.com)
- [Google Maps](https://developers.google.com/maps/documentation)
- [HERE Maps](https://developer.here.com/develop/rest-apis)
- [Waze](https://developers.google.com/waze)
- [Apple Maps MapKit JS](https://developer.apple.com/maps/mapkitjs)
- [OpenStreetMap](https://openstreetmap.org)
- [Trello](https://developers.trello.com)
- [Slack](https://api.slack.com)
- [Dialogflow](https://dialogflow.com)
- [Wit.ai](https://wit.ai)
- [Microsoft Bot Framework](https://dev.botframework.com)
- [Kata.ai Chatbot](https://docs.kata.ai)
  - [Bot Studio Tutorial](https://docs.kata.ai/tutorial/bot-studio)

But must of them require us to register for an API Key and understanding of OAuth.

There are some easier alternatives for learning purpose:

- [JSONPlaceholder - Fake online REST API for developers](https://jsonplaceholder.typicode.com)
- [My JSON Server - Fake online REST server for teams](https://my-json-server.typicode.com)
- [Reqres - A hosted REST-API ready to respond to your AJAX requests](https://reqres.in)
- [REST Countries](https://restcountries.eu)
- [Star Wars API](https://swapi.co)
- [GitHub API](https://developer.github.com/v3)
- [Ghibli API](https://ghibliapi.herokuapp.com)
- [Company Logo API](https://blog.clearbit.com/logo)
  - [`logo.clearbit.com`](https://logo.clearbit.com)
